Bienvenue à la Mission Vaccination

Instructions

Bienvenue à la Mission Vaccination. Vous allez accomplir différentes missions pour aider à éduquer le public sur les vaccins.

Mission 1 : Démasquer les Fausses Informations

Vous avez découvert une publication en ligne contenant des informations douteuses sur les vaccins. Identifiez les affirmations fausses pour les signaler.





Mission 2 : Détecter les Théories de Conspiration

Identifiez les arguments couramment utilisés dans les théories de conspiration sur les vaccins.
